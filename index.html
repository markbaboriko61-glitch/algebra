Определение векторного пространства. Примеры векторных пространств

Определение. Множество V над полем K называется векторным (линейным) пространством, если на нём заданы две операции — сложение + : V×V → V и умножение на скаляр · : K×V → V — удовлетворяющие 8 аксиомам.

Аксиомы сложения («V — абелева группа»):

  1. коммутативность: u + v = v + u;
  2. ассоциативность: (u + v) + w = u + (v + w);
  3. существование нуля 0: v + 0 = v;
  4. существование противоположного −v: v + (−v) = 0.

Аксиомы умножения на скаляр:

  1. 1·v = v;
  2. α(βv) = (αβ)v;
  3. (α + β)v = αv + βv (дистрибутивность по скалярам);
  4. α(u + v) = αu + αv (дистрибутивность по векторам).

Примеры векторных пространств:

  • Kⁿ — столбцы (или строки) из n чисел; в частности ℝ², ℝ³ — плоскость и пространство «стрелок»;
  • M_{m×n}(K) — матрицы фиксированного размера;
  • K[x] — многочлены; K[x]_{≤n} — многочлены степени ≤ n;
  • C[a,b] — непрерывные функции на отрезке;
  • любое поле K над самим собой; нулевое пространство {0}.
💡
Простыми словами

Векторное пространство — это «мир стрелок», где можно делать две вещи: складывать стрелки и растягивать/сжимать их в любое число раз. 8 аксиом — это просто «правила честной игры»: порядок сложения не важен, есть ноль, есть обратный и т.д. Примеры: обычные координаты на плоскости, матрицы, многочлены, функции — всё это можно складывать и умножать на число.

Простейшие свойства векторных пространств

Из аксиом выводятся базовые свойства (и важно уметь их доказывать):

  1. Единственность нуля. Если 0₁ и 0₂ — нули, то 0₁ = 0₁ + 0₂ = 0₂.
  2. Единственность противоположного. Оно определено однозначно.
  3. 0·v = 0 (скалярный ноль даёт нулевой вектор): из 0·v = (0+0)v = 0·v + 0·v.
  4. α·0 = 0 для любого скаляра α.
  5. (−1)·v = −v: из v + (−1)v = (1 + (−1))v = 0·v = 0.
  6. Закон отсутствия делителей нуля: если α·v = 0, то α = 0 или v = 0.
  7. Корректность вычитания: u − v := u + (−v), уравнение x + v = u имеет единственное решение.
💡
Простыми словами

Это «очевидные» факты, которые на самом деле нужно вывести из правил: ноль только один, у каждого вектора ровно один «минус», умножение на 0 даёт ноль, а умножение на −1 «разворачивает» стрелку. Главное: если произведение числа на вектор равно нулю — значит, либо число ноль, либо вектор нулевой.

Линейные комбинации системы векторов. Линейная зависимость и линейная независимость. Примеры

Линейная комбинация векторов v₁,…,v_k — это выражение α₁v₁ + … + α_k v_k со скалярами α_i ∈ K. Комбинация называется тривиальной, если все α_i = 0.

Линейная зависимость. Система линейно зависима, если существует нетривиальная комбинация, равная нулю: α₁v₁ + … + α_k v_k = 0 при не всех нулевых α_i.

Линейная независимость. Система линейно независима, если равенство α₁v₁ + … + α_k v_k = 0 возможно только при α₁ = … = α_k = 0.

Примеры.

  • e₁=(1,0), e₂=(0,1) в ℝ² — линейно независимы.
  • (1,2) и (2,4) — линейно зависимы (второй = 2×первый).
  • Любая система, содержащая нулевой вектор или два одинаковых вектора — линейно зависима.
  • 1, x, x² в K[x] — линейно независимы.
💡
Простыми словами

Линейная комбинация — это «смесь» векторов с коэффициентами. Зависимость означает «один лишний»: какой-то вектор можно собрать из остальных. Независимость — «никто не лишний»: ноль можно собрать только взяв все коэффициенты нулём. Проверка простая: «можно ли один вектор выразить через другие?»

Критерии и свойства линейной зависимости

Критерий (главный). Система из k ≥ 2 векторов линейно зависима ⇔ хотя бы один из векторов линейно выражается через остальные.

Свойства:

  1. Система с нулевым вектором всегда линейно зависима.
  2. Один вектор v линейно зависим ⇔ v = 0.
  3. Если подсистема линейно зависима, то и вся система линейно зависима.
  4. Любая подсистема линейно независимой системы линейно независима.
  5. Если система линейно независима, а после добавления вектора u стала зависимой, то u единственным образом выражается через остальные.

Матричный критерий. В Kⁿ система из n векторов линейно независима ⇔ det ≠ 0; в общем случае — ранг матрицы из векторов равен их числу.

💡
Простыми словами

Главный признак зависимости: кто-то из векторов «повторяет» других — его можно собрать из остальных. Полезные правила: если есть ноль — сразу зависимы; если кусок системы зависим — вся зависима; если векторов больше, чем размерность, — точно зависимы. Для чисел удобно считать определитель или ранг.

Подпространство. Примеры. Линейная оболочка системы векторов

Определение. Непустое подмножество U ⊆ V — подпространство, если оно замкнуто относительно сложения и умножения на скаляр.

Критерий подпространства. U — подпространство ⇔ U ≠ ∅ и для любых u,w ∈ U, α,β ∈ K вектор αu + βw ∈ U. В частности 0 ∈ U.

Примеры:

  • {0} и само V — тривиальные подпространства;
  • прямые и плоскости через 0 в ℝ³;
  • решения однородной СЛУ Ax = 0;
  • многочлены степени ≤ n внутри K[x].

Линейная оболочка. ⟨v₁,…,v_k⟩ = множество всех линейных комбинаций этих векторов. Это наименьшее подпространство, содержащее все v_i.

💡
Простыми словами

Подпространство — это «мир внутри мира»: меньшее пространство, в котором по-прежнему можно складывать и умножать, не выпадая наружу. Главная проверка: «не вылетаем ли мы наружу, складывая или растягивая?». Обязательно содержит ноль. Оболочка — это «всё, что можно собрать» из данных векторов.

6↑ наверх

Эквивалентные системы векторов, их свойства

Определение. Две системы векторов эквивалентны, если каждый вектор одной линейно выражается через другую и наоборот. Эквивалентность ⇔ обе порождают одну и ту же линейную оболочку.

Свойства:

  1. Это отношение эквивалентности: рефлексивно, симметрично, транзитивно.
  2. Эквивалентные системы имеют одинаковый ранг (максимальное число линейно независимых векторов).
  3. Любые две базы (максимальные линейно независимые подсистемы) эквивалентных систем равномощны.
💡
Простыми словами

Две системы эквивалентны, если «описывают одно и то же облако» — из одной можно собрать другую и обратно. Набор векторов может быть разным, но порождённое пространство и его размер (ранг) — одинаковые.

7↑ наверх

Элементарные преобразования системы векторов

Элементарные преобразования системы векторов:

  1. перестановка двух векторов;
  2. умножение вектора на ненулевой скаляр α ≠ 0;
  3. прибавление к одному вектору другого, умноженного на скаляр.

(Иногда добавляют вычёркивание нулевого вектора.)

Главное свойство. Элементарные преобразования не меняют линейную оболочку системы (получается эквивалентная система), а значит и ранг. Каждое обратимо.

Применение. Приводя систему к ступенчатому виду, находят ранг, базис оболочки и выражения зависимых векторов.

💡
Простыми словами

Это «разрешённые ходы», которыми можно перемешивать векторы, не меняя само облако, которое они порождают: переставить, растянуть, добавить один к другому. Именно ими работает метод Гаусса.

8↑ наверх

Лемма (теорема) Штейница

Теорема Штейница (о замене). Пусть система a₁,…,a_k линейно независима и каждый a_i линейно выражается через систему b₁,…,b_m. Тогда k ≤ m, и можно заменить k подходящих векторов b на векторы a так, что полученная система по-прежнему порождает ту же оболочку.

Следствия:

  1. В линейно независимой системе не может быть больше векторов, чем в любой порождающей системе.
  2. Все базисы пространства равномощны — отсюда корректность понятия размерности.

Это ключевая теорема: именно она позволяет корректно говорить о размерности пространства.

💡
Простыми словами

Смысл: линейно независимых векторов не может быть больше, чем образующих («нельзя впихнуть больше независимых направлений, чем есть »строительных блоков«). Именно поэтому у всех базисов одинаковое число векторов — это и есть размерность.

Система образующих векторного пространства. Базис векторного пространства. Примеры

Система образующих (порождающая). Система v₁,…,v_k порождает V, если ⟨v₁,…,v_k⟩ = V, то есть любой вектор есть их линейная комбинация.

Базис. Базис — это линейно независимая система образующих. Эквивалентные определения: это максимальная линейно независимая система, либо минимальная порождающая.

Основное свойство. e₁,…,e_n — базис ⇔ каждый вектор единственным образом разлагается: v = α₁e₁ + … + α_n e_n.

Примеры:

  • стандартный базис e₁,…,e_n в Kⁿ;
  • 1, x, x², …, xⁿ в K[x]_{≤n};
  • матричные единицы E_{ij} в M_{m×n}(K).
💡
Простыми словами

Образующие — это «строительный набор», из которого можно собрать всё пространство (но может быть с лишними деталями). Базис — это «идеальный набор»: ровно столько деталей, сколько нужно, ничего лишнего. В нём каждый вектор собирается ровно одним способом — это и есть координаты.

10↑ наверх

Размерность векторного пространства, подпространства

Определение. Размерность dim V — это число векторов в любом базисе (оно корректно по теореме Штейница — все базисы равномощны).

Примеры: dim Kⁿ = n; dim K[x]_{≤n} = n+1; dim M_{m×n} = mn; dim {0} = 0. Если базис бесконечен — пространство бесконечномерно (например, K[x]).

Размерность подпространства. Если U ⊆ V и dim V = n, то dim U ≤ n, причём dim U = n ⇔ U = V. Любой базис подпространства можно дополнить до базиса всего пространства (теорема о дополнении).

💡
Простыми словами

Размерность — это «число степеней свободы»: сколько независимых направлений есть в пространстве. У плоскости — 2, у обычного пространства — 3. У подпространства степеней свободы не больше, чем у всего пространства; и если их ровно столько же — это уже всё пространство.

11↑ наверх

Свойства n-мерных векторных пространств

Пусть dim V = n. Тогда:

  1. Любые n+1 и более векторов линейно зависимы.
  2. Любая линейно независимая система из n векторов — базис.
  3. Любая порождающая система из n векторов — базис.
  4. Любую линейно независимую систему можно дополнить до базиса.
  5. Из любой порождающей системы можно выделить базис.
  6. Каждое подпространство конечномерно и dim U ≤ n.

Эти свойства — прямые следствия теоремы Штейница.

💡
Простыми словами

Если размерность равна n, то ровно n «правильных» векторов хватает, чтобы быть базисом, а больше n векторов всегда «лишние». Нехватающие можно добавить, лишние — убрать.

12↑ наверх

Координаты вектора

Определение. Если e₁,…,e_n — базис, то каждый вектор единственным образом записывается v = x₁e₁ + … + x_n e_n. Числа (x₁,…,x_n) — координаты вектора в данном базисе.

Свойства. При сложении векторов координаты складываются, при умножении на скаляр — умножаются. Именно поэтому соответствие v ↦ (x₁,…,x_n) задаёт изоморфизм V ≅ Kⁿ.

Координаты зависят от выбора базиса; при смене базиса они пересчитываются через матрицу перехода (вопрос 13).

💡
Простыми словами

Координаты — это «рецепт» вектора: сколько какого базисного вектора взять. Рецепт всегда единственный. Благодаря этому любое n-мерное пространство можно «перевести» в обычные столбцы чисел. Поменяешь базис — поменяются и координаты.

13↑ наверх

Матрица перехода от одного базиса к другому, преобразование координат

Матрица перехода. Пусть e = (e₁,…,e_n) и e' = (e'₁,…,e'_n) — два базиса. Матрица перехода T состоит из столбцов — координат новых базисных векторов e'_j в старом базисе: e'_j = ∑_i T_{ij} e_i, то есть e' = e·T.

Свойства. T обратима (det T ≠ 0), т.к. столбцы — координаты базиса.

Преобразование координат. Если x — столбец координат в старом базисе, x' — в новом, то x = T x', откуда x' = T⁻¹ x. Обратите внимание: базисы и координаты преобразуются «встречно» (одни через T, другие через T⁻¹).

💡
Простыми словами

Матрица перехода — это «словарь-переводчик» между двумя базисами: в её столбцах написано, как новые базисные векторы выражаются через старые. Чтобы пересчитать координаты, умножают на эту матрицу (или на обратную).

14↑ наверх

Изоморфизмы векторных пространств

Определение. Отображение f : V → W — изоморфизм, если оно линейно (f(αu+βv)=αf(u)+βf(v)) и биективно. Пространства V и W называются изоморфными (V ≅ W).

Пример. Выбор базиса даёт изоморфизм V ≅ Kⁿ (вектор ↦ столбец координат). Например, K[x]_{≤n} ≅ K^{n+1}.

Обратное. Обратное отображение f⁻¹ тоже линейно, так что это симметричное отношение.

💡
Простыми словами

Изоморфизм — это «идеальный переводчик» между двумя пространствами: он сохраняет сложение и растяжение и переводит одно в другое без потерь. Если два пространства изоморфны, они «устроены одинаково», хотя элементы могут выглядеть по-разному.

15↑ наверх

Свойства изоморфизма векторных пространств

Пусть f : V → W — изоморфизм. Тогда:

  1. f(0) = 0, f(−v) = −f(v).
  2. Образ линейно независимой системы линейно независим, образ зависимой — зависим.
  3. Образ базиса — базис; поэтому dim V = dim W.
  4. Образ подпространства — подпространство той же размерности.
  5. Композиция изоморфизмов и обратное отображение — снова изоморфизмы.

Отношение изоморфности рефлексивно, симметрично и транзитивно.

💡
Простыми словами

Изоморфизм не портит «структуру»: независимые остаются независимыми, базис переходит в базис, размерность сохраняется. Поэтому «перевод» идёт в обе стороны и ничего не теряется.

16↑ наверх

Критерий изоморфизма векторных пространств

Критерий. Два конечномерных пространства над одним полем изоморфны ⇔ их размерности равны: V ≅ W ⇔ dim V = dim W.

Доказательство (идея). Необходимость — из свойства «образ базиса есть базис». Достаточность: если dim V = dim W = n, выберем базисы e_i в V и f_i в W и положим e_i ↦ f_i; распространив по линейности, получим изоморфизм.

Следствие. Любое n-мерное пространство над K изоморфно Kⁿ — «эталонной модели».

💡
Простыми словами

Главный вывод: размерность полностью определяет пространство с точностью до изоморфизма. Два пространства одной размерности — это «одно и то же по сути». Поэтому все n-мерные пространства можно изучать на примере обычных столбцов Kⁿ.

17↑ наверх

Базис и ранг системы векторов

Базис системы векторов — это максимальная линейно независимая подсистема. Ранг системы — число векторов в таком базисе, т.е. rk = dim⟨v₁,…,v_k⟩.

Свойства:

  1. Все базисы системы содержат одинаковое число векторов (по Штейницу).
  2. Ранг не меняется при элементарных преобразованиях и у эквивалентных систем одинаков.
  3. Остальные векторы линейно выражаются через базисные.

Нахождение. Записывают векторы строками матрицы и приводят к ступенчатому виду: ненулевые строки дают базис, их число — ранг.

💡
Простыми словами

Ранг — это «сколько по-настоящему разных направлений» в наборе векторов, а базис — сам такой минимальный набор. Остальные векторы — повторы, собираемые из базисных. На практике считается методом Гаусса.

18↑ наверх

Наличие ненулевого минора в линейно независимых строках матрицы. Линейная независимость строк, проходящих через ненулевой минор

Минор порядка r — определитель подматрицы на пересечении r строк и r столбцов.

Утверждение 1. Если строки матрицы линейно независимы, то среди них есть ненулевой минор максимального порядка (порядка, равного числу строк).

Утверждение 2 (обратное). Если некоторый минор порядка r отличен от нуля, то r строк, проходящих через этот минор, линейно независимы (и тоже самое для столбцов).

Эти два факта — основа теоремы о ранге (вопрос 19): ранг матрицы равен порядку наибольшего ненулевого минора.

💡
Простыми словами

Минор — это определитель «вырезанного квадратика» из матрицы. Если он не ноль, то строки (и столбцы), через которые он проходит, точно независимы. Поэтому «самый большой ненулевой квадратик» и показывает ранг.

19↑ наверх

Ранг матрицы. Теорема о ранге матрицы

Для матрицы определяют три величины:

  • строчный ранг — размерность оболочки строк;
  • столбцовый ранг — размерность оболочки столбцов;
  • минорный ранг — порядок наибольшего ненулевого минора.

Теорема о ранге. Все три ранга равны между собой. Это общее число и называется рангом матрицы rk A.

Следствия: rk A = rk Aᵀ; rk A ≤ min(m, n); ранг не меняется при элементарных преобразованиях строк и столбцов.

💡
Простыми словами

Главная мысль: сколько независимых строк, столько же и независимых столбцов — и это же число равно размеру самого большого ненулевого минора. Это и есть ранг — одно число, характеризующее матрицу.

20↑ наверх

Методы вычисления ранга матрицы

1. Метод элементарных преобразований (Гаусс). Приводят матрицу к ступенчатому виду; число ненулевых строк (ведущих элементов) равно рангу. Это самый эффективный способ.

2. Метод окаймляющих миноров. Ищут ненулевой минор, затем «окаймляют» его: если все окаймляющие миноры порядка r+1 равны нулю, то rk = r.

3. По определению — порядок наибольшего ненулевого минора (громоздко для больших матриц).

На практике почти всегда используют метод Гаусса.

💡
Простыми словами

Ранг удобнее всего считать методом Гаусса: ступеньками обнуляем низ, и сколько ненулевых строк осталось — таков и ранг. Можно и через миноры, но это дольше.

21↑ наверх

Ранг суммы и произведения матриц

Ранг суммы: rk(A + B) ≤ rk A + rk B.

Ранг произведения: rk(AB) ≤ min(rk A, rk B).

Неравенство Сильвестра: если A размера m×n, B размера n×p, то rk A + rk B − n ≤ rk(AB).

Инвариантность. Умножение на обратимую матрицу не меняет ранг: rk(PA) = rk(AQ) = rk A при обратимых P, Q. Это потому, что умножение на обратимую матрицу равносильно элементарным преобразованиям.

💡
Простыми словами

Интуиция: «ранг — это богатство направлений». При сложении направления не могут «родиться из ничего» — поэтому ранг суммы не больше суммы рангов. При умножении ранг не может вырасти выше, чем у множителей. А умножение на обратимую (невырожденную) вообще ранг не меняет.

22↑ наверх

Критерий совместности системы линейных уравнений

Теорема Кронекера – Капелли. СЛУ Ax = b совместна (имеет решение) ⇔ ранг основной матрицы равен рангу расширенной: rk A = rk(A | b).

Количество решений. Пусть r = rk A = rk(A|b), n — число неизвестных:

  • r = n — единственное решение;
  • r < n — бесконечно много решений с n − r свободными параметрами;
  • rk A < rk(A|b) — решений нет (несовместна).

Смысл. Система совместна тогда и только тогда, когда столбец b линейно выражается через столбцы A.

💡
Простыми словами

Система имеет решение только если правая часть b «собирается» из столбцов матрицы. Проверяют это сравнением двух рангов: без b и с b. Равны — решение есть; ранг вырос — решений нет.

23↑ наверх

Однородные и неоднородные системы линейных уравнений. Метод Гаусса решения систем линейных уравнений

Однородная СЛУ: Ax = 0 (правая часть нулевая). Всегда совместна (есть нулевое решение).

Неоднородная СЛУ: Ax = b, b ≠ 0.

Метод Гаусса. Расширенную матрицу (A | b) элементарными преобразованиями строк приводят к ступенчатому виду (прямой ход), затем обратным ходом выражают базисные неизвестные через свободные.

Этапы: определить ранги (совместность), выбрать базисные/свободные переменные, выписать общее решение. Преимущество — работает всегда и полиномиально быстро.

💡
Простыми словами

Однородная — это система «= 0», у неё всегда есть хотя бы нулевое решение. Неоднородная — с ненулевой правой частью. Метод Гаусса — это аккуратное «выражаем одни переменные через другие» ступеньками, пока система не станет прозрачной.

24↑ наверх

Однородные системы, условие существования нетривиального решения. Фундаментальная система решений

Множество решений. Множество решений однородной СЛУ Ax = 0 с n неизвестными — это подпространство в Kⁿ размерности n − rk A (дефект).

Условие нетривиального решения. Нетривиальное решение существует ⇔ rk A < n. Для квадратной матрицы это ⇔ det A = 0. В частности, если уравнений меньше, чем неизвестных, нетривиальное решение есть всегда.

ФСР. Фундаментальная система решений — это базис пространства решений (n − r линейно независимых решений). Общее решение — любая их линейная комбинация.

Нахождение ФСР: приводят к ступенчатому виду, придают свободным переменным поочерёдно значения из единичной матрицы.

💡
Простыми словами

Если независимых уравнений меньше, чем неизвестных, то появляются «свободные» переменные и решений бесконечно много. ФСР — это «набор кирпичиков», из которых складываются все решения. Их ровно «число неизвестных минус ранг».

25↑ наверх

Связь решений однородной и неоднородной систем

Теорема о структуре решения. Общее решение неоднородной СЛУ Ax = b имеет вид x = x₀ + x_одн, где x₀ — любое (частное) решение неоднородной системы, а x_одн — общее решение соответствующей однородной Ax = 0.

Геометрически: множество решений неоднородной системы — это сдвинутое (аффинное) подпространство: подпространство решений однородной, сдвинутое на вектор x₀.

Следствие. Разность двух решений неоднородной системы — всегда решение однородной.

💡
Простыми словами

Решения неоднородной системы = одно любое решение + все решения «нулевой» системы. Представь облако решений однородной системы, целиком сдвинутое на один найденный ответ. Поэтому достаточно найти одно решение и всю ФСР.

26↑ наверх

Задание подпространства векторного пространства системой линейных уравнений

Подпространство можно задавать двумя способами:

  • Параметрически — как линейную оболочку ⟨a₁,…,a_k⟩ («изнутри»).
  • Неявно — как множество решений однородной системы Ax = 0 («извне», ограничениями).

Связь. Подпространство U размерности d в Kⁿ задаётся n − d независимыми линейными уравнениями.

Переход от оболочки к уравнениям. Записывают векторы a_i строками, ищут левое ядро (решения yᵀ a_i = 0 для всех i): эти y и дают коэффициенты искомых уравнений (см. практику, задача П2).

💡
Простыми словами

Подпространство можно описать двумя способами: либо «из каких векторов оно собирается» (оболочка), либо «каким уравнениям оно удовлетворяет» (ограничения). Переход между ними — это решение системы: число уравнений = размерность пространства минус размерность подпространства.

27↑ наверх

Операции над подпространствами

Пусть U, W ⊆ V — подпространства.

  • Пересечение U ∩ W — всегда подпространство (векторы, лежащие и в U, и в W).
  • Сумма U + W = {u + w : u ∈ U, w ∈ W} — наименьшее подпространство, содержащее и U, и W.

Важное замечание. Объединение U ∪ W, вообще говоря, не подпространство (сумма двух векторов из разных может выпасть). Именно поэтому вводят сумму U + W — «оболочку объединения».

Эти операции связаны формулой Грассмана (вопрос 28).

💡
Простыми словами

Два подпространства можно «пересекать» (берём общее) и «складывать» (берём всё, что можно собрать из обоих). Просто «сложить множества» (объединить) нельзя — получится не подпространство.

28↑ наверх

Размерности суммы и пересечения подпространств

Формула Грассмана:

dim(U + W) = dim U + dim W − dim(U ∩ W)

Идея доказательства. Берём базис c₁,…,c_p пересечения U ∩ W, дополняем его до базиса U (добавляя a_i) и отдельно до базиса W (добавляя b_j). Тогда {c, a, b} — базис U + W, откуда и следует формула.

Следствие. Если U ∩ W = {0}, то dim(U + W) = dim U + dim W — это прямая сумма (вопрос 29).

💡
Простыми словами

Это как формула включений-исключений для размерностей: складываем размеры двух подпространств, но вычитаем общую часть, чтобы не считать её дважды. Если общего нет (только ноль) — размеры просто складываются.

29↑ наверх

Прямая сумма подпространств. Критерии

Определение. Сумма U + W называется прямой (обозначается U ⊕ W), если каждый вектор суммы единственным образом представляется в виде u + w.

Критерии прямоты (равносильны):

  1. U ∩ W = {0};
  2. представление нуля единственно: 0 = u + w ⇒ u = w = 0;
  3. dim(U + W) = dim U + dim W;
  4. объединение базисов U и W даёт базис суммы.

Обобщение. Для нескольких слагаемых U₁ + … + U_k прямая ⇔ каждое U_i пересекает сумму остальных только по нулю.

💡
Простыми словами

Прямая сумма — это когда два подпространства «не пересекаются лишним» (только в нуле), и поэтому каждый вектор разлагается на части ровно одним способом. Как разложить вектор на «x-часть» и «y-часть» — и только так.

30↑ наверх

Прямое дополнение

Определение. Подпространство W называется прямым дополнением к U в V, если V = U ⊕ W, то есть U ∩ W = {0} и U + W = V.

Теорема существования. Для любого подпространства U конечномерного V прямое дополнение существует (берём базис U, дополняем его до базиса V; оболочка добавленных векторов и есть W).

Важно: прямое дополнение не единственно (их обычно много), но все они имеют одну размерность dim V − dim U. Нахождение — см. практику.

💡
Простыми словами

Прямое дополнение — это «недостающая половинка»: подпространство, которое вместе с данным даёт всё пространство и не пересекается с ним. Таких «половинок» может быть много, но размер у них один и тот же.

31↑ наверх

Линейные отображения векторных пространств. Примеры

Определение. Отображение f : V → W называется линейным, если оно сохраняет операции:

  • аддитивность: f(u + v) = f(u) + f(v);
  • однородность: f(αv) = α f(v).

Эквивалентно одним условием: f(αu + βv) = αf(u) + βf(v). Линейное отображение из V в себя называют линейным оператором.

Примеры:

  • нулевое f(v)=0 и тождественное f(v)=v;
  • умножение на матрицу x ↦ Ax в Kⁿ;
  • поворот, проекция, гомотетия v ↦ λv;
  • дифференцирование p ↦ p' в K[x].

Не линейны: f(v)=v+v₀ (сдвиг, если v₀≠0), f(x)=x².

💡
Простыми словами

Линейное отображение — это «честное» преобразование: оно не ломает сложение и растяжение (сначала сложить, потом преобразовать — то же самое). Проверка на экзамене: подставь сумму и число-множитель и посмотри, выносятся ли они. Сдвиг и квадрат — не линейны.

32↑ наверх

Простейшие свойства линейных отображений

Пусть f : V → W линейно. Тогда:

  1. f(0) = 0 и f(−v) = −f(v).
  2. f(α₁v₁ + … + α_k v_k) = α₁ f(v₁) + … + α_k f(v_k) — образ линейной комбинации есть комбинация образов.
  3. Образ линейно зависимой системы линейно зависим (но образ независимой может стать зависимым — если есть ядро).
  4. Отображение полностью определяется своими значениями на базисе.
  5. Композиция линейных отображений линейна.

Эти свойства показывают, что линейное отображение «живёт» согласованно со структурой пространства.

💡
Простыми словами

Главное: линейное отображение нуль переводит в нуль и сохраняет любые «смеси» векторов. Поэтому, чтобы задать всё отображение, достаточно сказать, куда идут базисные векторы — остальное восстановится само.

33↑ наверх

Образ и прообраз подпространства

Пусть f : V → W линейно.

Образ подпространства U ⊆ V: f(U) = {f(u) : u ∈ U}. Это подпространство в W, причём dim f(U) ≤ dim U.

Прообраз подпространства S ⊆ W: f⁻¹(S) = {v ∈ V : f(v) ∈ S}. Это подпространство в V, содержащее ядро Ker f = f⁻¹({0}).

Свойства: образ подпространства — подпространство; прообраз подпространства — подпространство; dim f⁻¹(S) = dim(S ∩ Im f) + dim Ker f.

💡
Простыми словами

Образ — это «куда попадают» векторы из набора после преобразования. Прообраз — «откуда приходят» векторы, попадающие в заданное множество. Оба остаются подпространствами.

34↑ наверх

Теорема о продолжении отображения базиса

Теорема. Пусть e₁,…,e_n — базис V, а w₁,…,w_n — произвольные векторы в W. Тогда существует и притом единственное линейное отображение f : V → W с f(e_i) = w_i для всех i.

Формула: f(x₁e₁ + … + x_n e_n) = x₁ w₁ + … + x_n w_n.

Следствия: линейное отображение однозначно задаётся значениями на базисе; именно это позволяет задавать оператор матрицей (вопрос 36).

💡
Простыми словами

Смысл: чтобы задать линейное отображение, достаточно «разложить по полочкам» базис — решить, куда отправится каждый базисный вектор. Причём отправить можно куда угодно, и такое отображение будет ровно одно.

35↑ наверх

Действия над линейными отображениями

На множестве линейных отображений Hom(V, W) вводят операции:

  • сумма: (f + g)(v) = f(v) + g(v);
  • умножение на скаляр: (αf)(v) = α·f(v);
  • композиция: (g ∘ f)(v) = g(f(v)) для f: V→W, g: W→U.

Свойства. Относительно сложения и умножения на скаляр Hom(V, W) само является векторным пространством. Композиция ассоциативна и дистрибутивна относительно сложения, но в общем некоммутативна.

Этим операциям соответствуют сложение и умножение матриц (вопрос 38).

💡
Простыми словами

Линейные отображения можно складывать, умножать на число и сцеплять друг за другом (композиция — это «сначала одно, потом другое»). Порядок в композиции важен. Всё это зеркалит действия с матрицами.

36↑ наверх

Матрица линейного оператора

Определение. Пусть φ : V → V — линейный оператор, e₁,…,e_n — базис. Разложим φ(e_j) = ∑_i a_{ij} e_i. Числа a_{ij} образуют матрицу оператора A = [φ]_e: в j-м столбце стоят координаты образа φ(e_j).

Главное соотношение. Если x — столбец координат вектора, то координаты образа равны Ax. Таким образом, действие оператора в координатах — это умножение на матрицу.

Соответствие. Выбор базиса задаёт взаимно однозначное соответствие между операторами и матрицами n×n.

💡
Простыми словами

Матрица оператора — это «инструкция» в выбранном базисе: в каждом столбце записано, во что превращается очередной базисный вектор. Тогда действие на любой вектор — это просто «умножить на матрицу».

37↑ наверх

Изменение матрицы линейного оператора при переходе к новому базису

Формула подобия. Пусть A = [φ]_e и A' = [φ]_{e'} — матрицы одного оператора в базисах e и e', а T — матрица перехода от e к e'. Тогда:

A' = T⁻¹ A T

Подобие. Две матрицы, связанные таким соотношением, называются подобными. Подобные матрицы описывают один и тот же оператор в разных базисах.

Инварианты. У подобных матриц совпадают определитель, след, ранг, характеристический и минимальный многочлены, собственные значения.

💡
Простыми словами

Один и тот же оператор в разных базисах выглядит разными матрицами, но они связаны формулой T⁻¹AT («перевёл координаты, применил, перевёл обратно»). Такие матрицы — подобные, и у них совпадают «паспортные» характеристики: определитель, след, собственные числа.

38↑ наверх

Матрица суммы и композиции линейных операторов

Пусть в фиксированном базисе операторы φ, ψ имеют матрицы A, B. Тогда:

  • сумма: [φ + ψ] = A + B;
  • умножение на скаляр: [αφ] = αA;
  • композиция: [φ ∘ ψ] = A·B (произведение матриц).

Следствие. Соответствие φ ↦ [φ] является изоморфизмом алгебр: оно сохраняет и линейную структуру, и умножение. Именно поэтому «некоммутативность композиции» отражает некоммутативность произведения матриц.

💡
Простыми словами

Сложению операторов соответствует сложение матриц, а «сначала один, потом другой» (композиции) — произведение матриц. Именно поэтому матрицы вообще были придуманы так, а не иначе.

39↑ наверх

Ядро и образ линейного отображения. Ранг и дефект

Пусть f : V → W линейно.

  • Ядро Ker f = {v ∈ V : f(v) = 0} — подпространство в V.
  • Образ Im f = {f(v) : v ∈ V} — подпространство в W.
  • Ранг rk f = dim Im f; дефект def f = dim Ker f.

Критерии: f инъективно ⇔ Ker f = {0} (дефект 0); f сюръективно ⇔ Im f = W.

Эти два числа связаны теоремой о ранге и дефекте (вопрос 41).

💡
Простыми словами

Ядро — это «что схлопывается в ноль», образ — «куда вообще можно попасть». Ранг = размер образа, дефект = размер ядра. Если ядро пустое (только ноль) — отображение ничего не склеивает (инъективно).

40↑ наверх

Нахождение ядра линейного оператора

Метод. Пусть оператор φ имеет в базисе матрицу A. Тогда Ker φ — это множество решений однородной системы Ax = 0.

Алгоритм:

  1. записать матрицу A оператора;
  2. решить Ax = 0 методом Гаусса;
  3. фундаментальная система решений даёт базис ядра;
  4. def φ = dim Ker φ = n − rk A.

Образ. Базис Im φ — это максимальная линейно независимая подсистема столбцов A (или образы базисных векторов).

💡
Простыми словами

Чтобы найти ядро, решают систему «матрица × вектор = 0» методом Гаусса: её решения и есть ядро. Размер ядра = число переменных минус ранг. Образ же «собирается» из столбцов матрицы.

41↑ наверх

Теорема о ранге и дефекте линейного оператора

Теорема о ранге и дефекте. Для линейного отображения f : V → W с конечномерным V:

dim Ker f + dim Im f = dim V

то есть дефект + ранг = размерность области определения.

Идея доказательства. Берём базис ядра, дополняем его до базиса V; образы добавленных векторов образуют базис Im f.

Следствия: для оператора φ: V→V инъективность ⇔ сюръективность ⇔ обратимость (вопрос 44).

💡
Простыми словами

Самая важная формула: «что склеилось» (дефект) плюс «что выжило» (ранг) = всё, с чем начинали (размерность V). Отсюда: если оператор из пространства в себя ничего не склеивает, то он и на всё попадает — и наоборот.

42↑ наверх

Пространство линейных операторов и его связь с пространством матриц

Пространство операторов. Множество всех линейных операторов L(V) (обозн. также End(V)) относительно сложения и умножения на скаляр является векторным пространством.

Размерность. Если dim V = n, то dim L(V) = n².

Связь с матрицами. Фиксация базиса задаёт изоморфизм L(V) ≅ M_n(K) с пространством квадратных матриц. Этот изоморфизм сохраняет и умножение (композиция ↔ произведение матриц), то есть является изоморфизмом алгебр.

Базис. Операторы E_{ij} (матричные единицы) образуют базис из n² операторов.

💡
Простыми словами

Все линейные операторы на пространстве сами образуют пространство (их можно складывать и умножать на число). Если зафиксировать базис, каждый оператор — это n×n матрица, поэтому размерность равна n² (столько в матрице клеток). Это и был вопрос про «n в квадрате».

43↑ наверх

Автоморфизмы векторных пространств

Определения. Изоморфизм — это линейное биективное отображение V → W. Автоморфизм — это изоморфизм пространства на себя (обратимый линейный оператор).

Различие изоморфизма и просто биекции. Биекция — это любое взаимно однозначное соответствие множеств, она может не уважать сложение/умножение. Изоморфизм — это биекция, которая ещё и линейна (сохраняет структуру).

Группа. Множество всех автоморфизмов V относительно композиции образует группу GL(V) (общая линейная группа); в базисе ей соответствуют обратимые матрицы GL_n(K).

Факт. Два конечномерных пространства изоморфны ⇔ равны их размерности.

💡
Простыми словами

Изоморфизм — это «переименование» пространства без потери структуры: всё, что верно в одном, верно и в другом. Отличие от просто биекции: биекция просто «переставляет точки», а изоморфизм ещё и уважает сложение и растяжение. Автоморфизм — это «обратимое преобразование пространства в себя».

44↑ наверх

Условия обратимости линейного оператора

Критерий обратимости. Для оператора φ : V → V на конечномерном V следующие условия равносильны:

  1. φ обратим (есть φ⁻¹);
  2. φ инъективен: Ker φ = {0} (дефект 0);
  3. φ сюръективен: Im φ = V;
  4. rk φ = n (полный ранг);
  5. матрица A невырождена: det A ≠ 0;
  6. φ переводит (какой-то) базис в базис.

Важно. Именно в конечномерном случае «инъективность ⇔ сюръективность» (следствие теоремы о ранге и дефекте). Для бесконечномерных это неверно.

💡
Простыми словами

Оператор обратим, если он ничего не теряет и ничего не склеивает. На экзамене проверяют любой из эквивалентов: ядро = только ноль, ранг полный, определитель не нуль, базис переходит в базис. В конечномерном пространстве достаточно проверить что-то одно.

45↑ наверх

Инвариантные подпространства, матрица линейного оператора при наличии инвариантного подпространства

Определение. Подпространство U ⊆ V называется инвариантным относительно оператора φ, если φ(U) ⊆ U (оператор не выводит векторы из U).

Примеры: {0}, всё V, Ker φ, Im φ, любое собственное подпространство V_λ.

Блочный вид матрицы. Если U инвариантно и мы выбираем базис, первые векторы которого лежат в U, то матрица приобретает блочно-треугольный вид:

A = [[B, C], [0, D]]

где B — матрица ограничения φ на U. Если же всё пространство распадается в прямую сумму инвариантных подпространств, матрица становится блочно-диагональной.

💡
Простыми словами

Инвариантное подпространство — это «комната, из которой оператор не выпускает векторы». Если такая комната есть, можно выбрать базис так, чтобы матрица стала проще (с нулевым углом). А если всё распадается на такие комнаты — матрица становится блочной по диагонали.

46↑ наверх

Сумма и пересечение инвариантных подпространств

Теорема. Если U и W инвариантны относительно φ, то инвариантны и их сумма U + W, и пересечение U ∩ W.

Доказательство (схема).

  • Пересечение: если v ∈ U ∩ W, то φ(v) ∈ U (т.к. U инвариантно) и φ(v) ∈ W, значит φ(v) ∈ U ∩ W.
  • Сумма: если v = u + w, то φ(v) = φ(u) + φ(w) ∈ U + W.

Следствие. Система инвариантных подпространств замкнута относительно этих операций (образует решётку).

💡
Простыми словами

Если две «комнаты» не выпускают векторы наружу, то и их общая часть, и их «объединённое пространство» тоже не выпускают. Просто проверяем определение на сумме и пересечении.

47↑ наверх

Собственное значение и собственный вектор оператора

Определение. Ненулевой вектор v ≠ 0 называется собственным для оператора φ, если φ(v) = λv для некоторого скаляра λ. Число λ называется собственным значением.

Геометрический смысл. Оператор на собственном векторе действует просто как растяжение/сжатие (в λ раз), не меняя направления.

Характеристическое уравнение. λ является собственным значением ⇔ det(A − λE) = 0. Многочлен χ(λ) = det(A − λE) называется характеристическим.

💡
Простыми словами

Собственный вектор — это «особое направление», которое оператор не поворачивает, а только растягивает или сжимает. Насколько растягивает — это собственное значение λ. Ищут их через уравнение det(A − λE)=0.

48↑ наверх

Линейная независимость собственных векторов, принадлежащих попарно различным собственным значениям

Теорема. Собственные векторы v₁,…,v_k, отвечающие попарно различным собственным значениям λ₁,…,λ_k, линейно независимы.

Доказательство (индукция). Пусть α₁v₁ + … + α_k v_k = 0. Применим φ и вычтем исходное равенство, умноженное на λ_k; последнее слагаемое исчезает, а коэффициенты при остальных домножаются на (λ_i − λ_k) ≠ 0. По индукции все α_i = 0.

Следствие. Если оператор в n-мерном пространстве имеет n различных собственных значений, то он диагонализуем (есть базис из собственных векторов).

💡
Простыми словами

Векторы с разными собственными значениями «смотрят в разные стороны» и не могут быть выражены друг через друга — они независимы. А если разных собственных чисел столько же, сколько размерность — оператор приводится к диагональному виду.

49↑ наверх

Подпространство Vλ

Определение. Для собственного значения λ множество

V_λ = {v ∈ V : φ(v) = λv} = Ker(φ − λ·id)

называется собственным подпространством. Это подпространство (ядро оператора), состоящее из всех собственных векторов с данным λ и нуля.

Свойства:

  • V_λ инвариантно относительно φ;
  • геометрическая кратность dim V_λ не превосходит алгебраическую (кратность корня в характеристическом многочлене);
  • сумма собственных подпространств по разным λ — прямая.

Критерий диагонализуемости. Оператор диагонализуем ⇔ сумма размерностей всех V_λ равна n (геом. кратности совпадают с алгебраическими).

💡
Простыми словами

Vλ — это «собрание» всех векторов, которые оператор растягивает ровно в λ раз (плюс ноль). Оно показывает, «сколько независимых направлений» с данным растяжением. Если таких направлений в сумме хватает на всё пространство — оператор диагонализуется.

50↑ наверх

Нахождение собственных векторов и собственных значений

Алгоритм.

  1. Составить характеристический многочлен χ(λ) = det(A − λE).
  2. Найти его корни λ₁,…,λ_k — это собственные значения (с учётом кратности).
  3. Для каждого λ_i решить однородную систему (A − λ_i E)x = 0; её ненулевые решения — собственные векторы, а ФСР даёт базис V_{λ_i}.

Кратности. Сравнивая геометрическую кратность (dim V_λ) с алгебраической, определяют диагонализуемость. Пример полного разбора — в практике (задача П1, оператор проекции).

💡
Простыми словами

Сначала находят числа λ (решая уравнение det(A − λE)=0), а потом для каждого λ решают систему и находят сами векторы. Это два шага: «найди на сколько растягивает» и «найди, какие именно направления».

51↑ наверх

Теорема Гамильтона – Кэли. Характеристический многочлен оператора и матрицы

Характеристический многочлен. χ_A(λ) = det(A − λE) — многочлен степени n. Он не зависит от выбора базиса (инвариант подобия), поэтому говорят о характеристическом многочлене оператора.

Теорема Гамильтона – Кэли. Каждая квадратная матрица (оператор) является корнем своего характеристического многочлена:

χ_A(A) = 0

Применения: выражение обратной матрицы как многочлена от A; вычисление высоких степеней Aⁿ через младшие.

💡
Простыми словами

Характеристический многочлен — это многочлен, корни которого — собственные значения. Теорема Гамильтона–Кэли говорит удивительную вещь: если в этот многочлен «подставить» саму матрицу, получится ноль. Это позволяет выражать большие степени матрицы через маленькие.

52↑ наверх

Минимальный многочлен оператора и матрицы

Определение. Минимальный многочлен μ(t) оператора — это нормированный (со старшим коэффициентом 1) многочлен наименьшей степени, для которого μ(φ) = 0.

Свойства:

  • μ(t) делит любой многочлен, аннулирующий φ (в частности, делит характеристический χ);
  • μ и χ имеют одинаковые корни (собственные значения), но возможно с меньшими кратностями;
  • подобные матрицы имеют один и тот же μ.

Критерий диагонализуемости. Оператор диагонализуем ⇔ минимальный многочлен разлагается на различные линейные множители (нет кратных корней).

💡
Простыми словами

Минимальный многочлен — это самый «короткий» многочлен, который обнуляет матрицу. Он делит характеристический и хранит «самую важную» информацию об операторе. Если у него нет повторяющихся корней — оператор диагонализуется.

53↑ наверх

Определение и вычисление жордановой нормальной формы и канонического базиса

Жорданова клетка. Клетка J_k(λ) — это k×k матрица с λ на диагонали и единицами над ней.

Теорема о ЖНФ. Над алгебраически замкнутым полем (например ℂ) любой оператор в подходящем (жордановом, каноническом) базисе имеет блочно-диагональную матрицу из жордановых клеток; эта форма единственна с точностью до порядка клеток.

Алгоритм вычисления:

  1. найти собственные значения (корни χ);
  2. для каждого λ определить число и размеры клеток по рангам степеней (A − λE)ᵐ: число клеток размера ≥ m равно rk(A−λE)⁽ᵐ⁻¹⁾ − rk(A−λE)ᵐ;
  3. построить канонический базис из жордановых цепочек (векторы (A−λE)vₖ = vₖ₋₁).

ЖНФ — самая «почти диагональная» форма для недиагонализуемых операторов.

💡
Простыми словами

Шаг 2. Матрица проекции. P = E − (n nᵀ)/(n·n):

      ⎡  2/3  -1/3   1/3 ⎤
P =   ⎢ -1/3   2/3   1/3 ⎥
      ⎣  1/3   1/3   2/3 ⎦

Проверка «проектора»: P² = P — выполняется. ✓

Шаг 3. Образ. Im P = сама плоскость x + y − z = 0. Базис: (1, -1, 0) и (1, 0, 1). Значит rk P = 2 (ранг).

Шаг 4. Ядро. ker P = направление нормали ⟨(1, 1, -1)⟩. Дефект = 1. Проверка: 2 + 1 = 3 = dim ℝ³. ✓

Шаг 5. Собственные значения и векторы. Проектор имеет только два собственных значения:

  • λ = 1 (кратность 2): векторы на плоскости не меняются. Собственные векторы: (1, -1, 0), (1, 0, 1).
  • λ = 0 (кратность 1): нормаль «схлопывается» в 0. Собственный вектор: (1, 1, -1).

Диагонализируемость. Да: есть базис из собственных векторов, в нём матрица = diag(1, 1, 0). Любой проектор всегда диагонализируем.

Исследование матриц на подобие

📝
Условие

Как проверить, подобны ли две матрицы A и B (то есть B = S⁻¹ A S для некоторой обратимой S).

Необходимые условия (быстрые проверки). У подобных матриц совпадают:

  • характеристический многочлен det(A − λE) (а значит и собственные значения с кратностями);
  • след tr A, определитель det A, ранг.

Если хоть одна из этих величин различна — матрицы не подобны.

Полный критерий. Матрицы подобны ⇔ у них одинаковая жорданова нормальная форма (с точностью до порядка клеток) — над алгебраически замкнутым полем. Над любым полем эквивалентный инвариант — набор инвариантных множителей (каноническая форма Смита матрицы λE − A).

Практический алгоритм. Посчитать ЖНФ обеих матриц (собственные значения + размеры клеток через ранги (A − λE)^k); совпадают — подобны.

Вопросы устной части (краткие ответы)

Условие

Набор «ловушек» и коротких вопросов, которые спрашивали на экзамене.

1. Размерность пространства линейных операторов. В n-мерном пространстве dim L(V) = n². После фиксации базиса каждому оператору соответствует матрица n×n, а это изоморфизм L(V) ≅ M_n(K), размерность которого n².

2. Что зафиксировать для связи «операторы ↔ матрицы». Нужно зафиксировать базис пространства (один и тот же для «входа» и «выхода»). Без фиксированного базиса матрицы нет.

3. Доказать, является ли отображение линейным оператором. Проверяем два свойства: φ(u + v) = φ(u) + φ(v) и φ(αv) = αφ(v). Если оба выполнены — линейный; иначе приводим контрпример (например, наличие свободного члена или квадрата ломает линейность).

4. Изоморфизм против биекции. Биекция — просто взаимно однозначное соответствие. Изоморфизм — это биекция, которая ещё и сохраняет операции (линейна). Для конечномерных пространств: линейное биективное отображение автоматически изоморфизм.

5. Критерий обратимости оператора. φ обратим ⇔ ker φ = {0} ⇔ rk φ = n ⇔ det A ≠ 0 ⇔ φ переводит базис в базис.

6. Матрица оператора, которая не меняется при смене базиса. Это скалярная матрица λE (оператор λ·id). Причина: матрица не меняется ⇔ S⁻¹ A S = A для всех обратимых S ⇔ A коммутирует со всеми матрицами ⇔ A = λE.

7. Поле {0, 1} и единственный базис. {0, 1} — это поле из двух элементов 𝔽₂ (GF(2)), с арифметикой по модулю 2. Именно над 𝔽₂ базис одномерного пространства единствен: единственный ненулевой скаляр — это 1, поэтому вектор нельзя «перемасштабировать» и получить другой базис. Над любым более богатым полем вектор можно умножить на любой ненулевой скаляр — и базисов уже много.

О1↑ наверх

Векторные пространства

Определение. Векторным (линейным) пространством над полем K называется множество V с двумя операциями — сложением + : V×V → V и умножением на скаляр · : K×V → V, удовлетворяющими восьми аксиомам.

Аксиомы сложения (V — абелева группа):

  1. коммутативность: u + v = v + u;
  2. ассоциативность: (u + v) + w = u + (v + w);
  3. нулевой вектор: существует 0 ∈ V, что v + 0 = v;
  4. противоположный: для каждого v есть −v с v + (−v) = 0.

Аксиомы умножения на скаляр:

  1. 1·v = v (единица поля);
  2. α(βv) = (αβ)v (ассоциативность);
  3. (α + β)v = αv + βv (дистрибутивность по скалярам);
  4. α(u + v) = αu + αv (дистрибутивность по векторам).

Простейшие следствия: 0·v = 0, α·0 = 0, (−1)v = −v; нулевой вектор и противоположный единственны; если αv = 0, то α = 0 или v = 0.

Примеры: арифметическое пространство столбцов Kⁿ; многочлены K[x] (и K[x] степени ≤ n); матрицы M_{m×n}(K); непрерывные функции; решения однородной СЛУ; само поле K как пространство над собой.

Подполе и поле. Поле — это множество со сложением и умножением, где обе операции коммутативны, ассоциативны, есть 0 и 1, и каждый ненулевой элемент обратим (например ℚ, ℝ, ℂ или конечное поле F₂ = {0,1}).

О2↑ наверх

Базис, размерность

Линейная оболочка. Набор всех линейных комбинаций векторов ⟨v₁,…,v_k⟩ = {α₁v₁ + … + α_k v_k} называется их линейной оболочкой; это наименьшее подпространство, содержащее эти векторы.

Порождающая система. Система векторов порождает V, если её линейная оболочка совпадает со всем V.

Определение базиса. Базис — это линейно независимая порождающая система. Равносильно: каждый вектор V единственным образом представляется в виде линейной комбинации базисных векторов. Также базис — это максимальная линейно независимая система и минимальная порождающая.

Размерность. Все базисы конечномерного пространства содержат одинаковое число векторов; это число dim V называется размерностью. Корректность опирается на основную лемму: если система из m векторов линейно независима, а система из n векторов порождает пространство, то m ≤ n.

Свойства:

  • в n-мерном пространстве любые n+1 векторов линейно зависимы;
  • любые n линейно независимых векторов образуют базис;
  • любую линейно независимую систему можно дополнить до базиса (теорема о дополнении).

Примеры: стандартный базис e₁,…,e_n в Kⁿ (dim = n); мономы 1, x, …, xⁿ в многочленах степени ≤ n (dim = n+1).

О3↑ наверх

Подпространство векторного пространства

Определение. Подмножество U ⊆ V называется подпространством, если оно само является векторным пространством относительно тех же операций.

Критерий подпространства. Непустое U ⊆ V является подпространством ⇔ оно замкнуто относительно двух операций:

  1. если u, v ∈ U, то u + v ∈ U;
  2. если u ∈ U и α ∈ K, то αu ∈ U.

(Это можно объединить: αu + βv ∈ U для любых α,β ∈ K.) В частности, любое подпространство содержит нулевой вектор.

Примеры: {0} и всё V (тривиальные); прямая или плоскость через нуль в ℝ³; множество решений однородной СЛУ; ядро и образ линейного отображения; многочлены степени ≤ n внутри всех многочленов.

Операции. Пересечение U ∩ W и сумма U + W = {u + w} снова являются подпространствами (объединение — как правило, нет). Связь размерностей (формула Грассмана): dim(U + W) = dim U + dim W − dim(U ∩ W).

О4↑ наверх

Линейная (не)зависимость

Линейная комбинация. Выражение α₁v₁ + … + α_k v_k со скалярами α_i ∈ K. Она называется тривиальной, если все α_i = 0.

Линейная независимость. Система v₁,…,v_k линейно независима, если равенство α₁v₁ + … + α_k v_k = 0 возможно только при всех α_i = 0.

Линейная зависимость. Система линейно зависима, если существует нетривиальная линейная комбинация, равная нулю (хотя бы один α_i ≠ 0).

Критерий. Система линейно зависима ⇔ хотя бы один из векторов линейно выражается через остальные.

Свойства:

  • система, содержащая нулевой вектор или два одинаковых, зависима;
  • подсистема независимой системы независима; надсистема зависимой зависима;
  • один вектор независим ⇔ он ненулевой.

Проверка на практике. Записать векторы в матрицу и найти ранг: система независима ⇔ ранг равен числу векторов. Для n векторов в n-мерном пространстве это равносильно det ≠ 0.

О5↑ наверх

Координаты вектора, построение матрицы перехода

Координаты вектора. Если e₁,…,e_n — базис и v = x₁e₁ + … + x_n e_n, то набор (x₁,…,x_n) — координаты вектора в этом базисе. Они определены однозначно. При сложении векторов координаты складываются, при умножении на скаляр — умножаются (изоморфизм V ≅ Kⁿ).

Матрица перехода. Пусть e = (e₁,…,e_n) и e' = (e'₁,…,e'_n) — два базиса. Матрица перехода T от e к e' — это матрица, в j-м столбце которой стоят координаты нового базисного вектора e'_j в старом базисе: e'_j = ∑_i T_{ij} e_i.

Преобразование координат. Если x — столбец координат в старом базисе, а x' — в новом, то x = T x', откуда x' = T⁻¹ x.

Построение на практике. Чтобы найти T, разлагают каждый вектор нового базиса по старому и записывают коэффициенты по столбцам. Свойства: T невырождена (обратима); матрица перехода в обратную сторону равна T⁻¹.

О6↑ наверх

Ранг

Ранг системы векторов. Это максимальное число линейно независимых векторов в системе; равно размерности их линейной оболочки.

Ранг матрицы. Для матрицы A совпадают три величины (теорема о ранге):

  • строчечный ранг — макс. число независимых строк;
  • столбцовый ранг — макс. число независимых столбцов;
  • минорный ранг — порядок наибольшего ненулевого минора.

Все три равны между собой и называются рангом rk A.

Вычисление. Методом Гаусса приводят матрицу к ступенчатому виду; ранг равен числу ненулевых строк (ведущих элементов).

Свойства: rk(AB) ≤ min(rk A, rk B); rk(A + B) ≤ rk A + rk B; ранг не меняется при умножении на обратимую матрицу. Для оператора ранг = dim Im, и по теореме о ранге и дефекте rk + def = n.

О7↑ наверх

Линейные отображения, линейные операторы

Линейное отображение. Отображение f : V → W между векторными пространствами над одним полем, сохраняющее операции:

  • аддитивность: f(u + v) = f(u) + f(v);
  • однородность: f(αv) = α f(v);

эквивалентно: f(αu + βv) = αf(u) + βf(v).

Линейный оператор — линейное отображение пространства в себя φ : V → V.

Простейшие свойства: f(0) = 0; f(−v) = −f(v); образ линейной комбинации = комбинация образов. Линейное отображение однозначно задаётся значениями на базисе.

Матрица оператора. В фиксированном базисе оператору соответствует матрица A, в столбцах которой — координаты образов базисных векторов; тогда [φ(v)] = A[v]. При смене базиса A' = T⁻¹ A T (подобные матрицы).

Операции: сумма, умножение на скаляр и композиция; им соответствуют A + B, αA, AB. Пространство операторов имеет размерность n² и изоморфно M_n(K). Примеры: нулевой, тождественный, поворот, проекция, дифференцирование.

О8↑ наверх

Собственные векторы, собственные значения

Определение. Ненулевой вектор v ≠ 0 — собственный для оператора φ, если φ(v) = λv для некоторого λ ∈ K. Число λ — собственное значение. Оператор на таком векторе работает как растяжение/сжатие без смены направления.

Характеристическое уравнение. λ — собственное значение ⇔ det(A − λE) = 0. Многочлен χ(λ) = det(A − λE) — характеристический, его степень равна n, он не зависит от базиса.

Собственное подпространство. V_λ = Ker(φ − λ·id) — подпространство всех собственных векторов с данным λ (и нуля).

Свойства: собственные векторы с различными λ линейно независимы; геометрическая кратность dim V_λ не превосходит алгебраическую. Оператор диагонализуем ⇔ есть базис из собственных векторов ⇔ сумма всех dim V_λ равна n.

Нахождение. Решить det(A − λE) = 0, затем для каждого λ найти ФСР системы (A − λE)x = 0.

О9↑ наверх

Ядро, образ линейного отображения

Ядро. Для линейного f : V → W: Ker f = {v ∈ V : f(v) = 0} — подпространство в V.

Образ. Im f = {f(v) : v ∈ V} — подпространство в W.

Ранг и дефект. rk f = dim Im f, def f = dim Ker f.

Критерии: f инъективно ⇔ Ker f = {0}; f сюръективно ⇔ Im f = W.

Теорема о ранге и дефекте: dim Ker f + dim Im f = dim V. Следствие для оператора φ : V → V: инъективность ⇔ сюръективность ⇔ обратимость.

Нахождение. Если A — матрица оператора, то Ker — множество решений Ax = 0 (базис — ФСР), а Im порождён столбцами A (базис — максимальная независимая подсистема столбцов). Причём def f = n − rk A.

О10↑ наверх

Однородная система линейных уравнений, фундаментальная система решений

Однородная СЛУ. Система вида Ax = 0 (все правые части нулевые). Всегда совместна: имеет нулевое (тривиальное) решение x = 0.

Пространство решений. Множество решений — это подпространство в Kⁿ (именно Ker A). Его размерность равна n − r, где n — число неизвестных, r = rk A.

Нетривиальное решение. Существует ⇔ r < n (ранг меньше числа неизвестных); в частности, когда уравнений меньше, чем неизвестных.

Фундаментальная система решений (ФСР). Это базис пространства решений: набор из n − r линейно независимых решений, через которые линейно выражается любое решение.

Построение ФСР. Приводят A к ступенчатому виду, выбирают r главных и n − r свободных неизвестных; поочерёдно полагают одну свободную равной 1, остальные 0, и выражают главные.

Связь с неоднородной СЛУ. Общее решение неоднородной системы Ax = b = одно частное решение + общее решение однородной (т.е. линейная комбинация ФСР).
